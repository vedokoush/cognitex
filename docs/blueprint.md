# **App Name**: CogniTex

## Core Features:

- Mood Mirror: Mood scanner allows students to choose mood icons, providing cause insights, motivation state analysis and suggested activities.
- SkillQuest: AI Learning Game that generates flashcard battles based on subjects or concepts provided by the student, rewarding correct answers with damage dealing and XP gains to increase the difficulty, as a motivational tool.
- LearnPulse Dashboard: Dashboard displaying Motivation Score calculated by AI, streak counter, time spent learning and skill growth indicator.
- Cogni Assistant: Motivational AI assistant that appears when needed to explain concepts, answer prompts, celebrate wins, give quotes and check on mood. The assistant acts as an encouraging tool.
- Landing Page: A simple landing page with onboarding questionnaire to kickstart the learning and motivation journey for the students.

## Style Guidelines:

- Primary color: Vibrant blue (#4285F4) to reflect energy, learning, and trustworthiness.
- Background color: Light blue (#E3F2FD) to maintain a bright and clean aesthetic.
- Accent color: Green (#34A853), analogous to blue, providing contrast and signaling growth.
- Font: 'Poppins', a geometric sans-serif for a modern, youthful, non-academic feel.
- Use a custom set of emotion-based icons with a modern and youthful style.
- Maintain a clean, non-competitive UI layout that avoids performance comparison with no ranking systems.
- Employ subtle animations and transitions to make the learning experience interactive and engaging.
